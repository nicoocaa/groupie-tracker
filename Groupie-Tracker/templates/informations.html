Qui sommes-nous ?

"Les Cramptons" est une équipe de développement composée de 3 personnes :

- Noémie DUBLANC : Développeuse Front-End, Designer
- Nicolas PRIGENT : Développeur Back-end, pilote automobile HOT WHEELS
- Maxime ISIDORE : Développeur Front-end, Chef de projet

Quel est notre projet ?

Notre projet est de créer un site web permettant de suivre les concerts de vos artistes préférés. Vous pourrez ainsi suivre les dates de concerts, les lieux, les horaires, les prix, et même les membres des groupes. Vous pourrez également consulter les informations pratiques pour vous rendre à ces concerts.
